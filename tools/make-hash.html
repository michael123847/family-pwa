Hash-Generator – Einmalig ausführen

Öffne diese Datei lokal im Browser. Gib deine Passphrase ein und kopiere den Hash in src/config.js als EXPECTED_HASH_B64. Derselbe Hash muss als PWA_TOKEN auf dem lokalen Server gesetzt werden.

← Passphrase eingeben und «Hash generieren» klicken
Sicherheitshinweis: Verwende eine Passphrase mit mindestens 4 zufälligen Wörtern (~50 Bit Entropie). Der Hash wird mit PBKDF2 / 200 000 Iterationen / SHA-256 berechnet. Dieser Hash ist im öffentlichen JS-Code sichtbar — Sicherheit kommt aus der Kosten-Funktion + Passphrase-Stärke.
