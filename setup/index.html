Familien-PWA · Einmaliges Setup

Damit dein Gerät die lokalen Funktionen (TODO-Liste, Familienbild) nutzen kann, muss einmalig das Sicherheitszertifikat der Heim-CA installiert werden. Danach funktioniert alles automatisch, solange du im Heim-WLAN bist.

iOS / iPadOS

Zertifikat herunterladen (iOS)

  1. Tippe auf den Download-Button — Safari fragt, ob du das Profil installieren möchtest.
  2. Einstellungen → VPN & Geräteverwaltung → mkcert… → «Installieren».
  3. Wichtig: Einstellungen → Allgemein → Info → Zertifikatsvertrauenseinstellungen → Toggle für «mkcert …» aktivieren.
Ohne Schritt 3 funktioniert nichts. iOS installiert das Profil, vertraut ihm aber nicht automatisch. Es gibt keine sichtbare Fehlermeldung — der Browser schlägt einfach still fehl.

macOS

Zertifikat herunterladen

  1. Datei doppelklicken → Schlüsselbund-App öffnet.
  2. In den System-Schlüsselbund importieren (nicht «Anmeldung»).
  3. Zertifikat im Schlüsselbund doppelklicken → Vertrauen aufklappen → «Immer vertrauen» wählen → Schliessen & Passwort bestätigen.

Android

Zertifikat herunterladen

  1. Einstellungen → Sicherheit → Verschlüsselung & Anmeldedaten → «CA-Zertifikat installieren».
  2. Heruntergeladene rootCA.pem auswählen & bestätigen.
Hinweis: Android-Chrome zeigt für user-installierte CAs ggf. eine Sicherheitswarnung in der Adressleiste. Das ist erwartet und harmlos.

Windows

Zertifikat herunterladen

  1. Win+R → certmgr.msc → Enter.
  2. Vertrauenswürdige Stammzertifizierungsstellen → Zertifikate → Rechtsklick → Alle Aufgaben → Importieren.
  3. Heruntergeladene Datei auswählen, Speicherort bestätigen.
